## Завдання 4  
Створити моделі для **інтернет-магазину**, які відповідатимуть таким критеріям:  
- Модель повинна вміщати **всі поля, які ви вважаєте потрібними**.  
- Додати **зв’язок між моделями**.  
- Усі **стандартні поля повинні бути порожніми** (за замовчуванням).  
- Додати моделі до **щонайменше 5 продуктів**.
